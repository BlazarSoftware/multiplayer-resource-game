# UI Asset Licenses

## Included assets
- GUI Pro - Casual Game texture subset: `gui_pro_casual_ofl.txt`
- Cartoon GUI Pack texture subset: `cartoon_gui_bpreplay_ofl.txt`, `cartoon_gui_lilita_ofl.txt`
- Google Fonts:
  - Bree Serif (`assets/fonts/bree_serif_regular.ttf`) — `google_fonts_bree_serif_ofl.txt`
  - Fredoka (`assets/fonts/fredoka_one_regular.ttf`) — `google_fonts_fredoka_ofl.txt` (legacy)
  - Nunito (`assets/fonts/nunito_regular.ttf`, `assets/fonts/nunito_bold.ttf`) — `google_fonts_nunito_ofl.txt`

## Notes
- Bree Serif, Fredoka, and Nunito are sourced from the official Google Fonts repository.
- Active UI heading font is Bree Serif; Nunito is used for body/UI controls.
- Kenney UI Pack and a CC0 parchment tile can be added later to `assets/ui/textures/bars/` and `assets/ui/textures/paper/` with their license files.
